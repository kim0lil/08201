# 스프링 프레임워크

스프링의 전반적인 기능과 내용을 다루고 있습니다.

## 스프링이 관리하는 빈을 생성하는 기본적인 방법

스프링을 사용하기 위하여 스프링을 동작 시키고 빈을 생성하는 기본적인 방법을 기술합니다. 

### 스프링 설정 에너테이션

스프링의 설정 파일을 등록 하는 방법입니다.

#### ComponentScan

스프링 설정 중에 자동으로 빈과 설정 파일을 등록 하여 스프링을 초기화 하는 에너테이션입니다.

#### Configuration annotation

@Configuration은 스프링의 설정 클래스를 등록 하는 에너테이션으로 ComponentScan시에 스프링의 설정 클래스로 등록 됩니다.
